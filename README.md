# Journey of Seed
**A journey of a seed.**

Journey of Seed is a demonstration of building app from scratch to most possible way.

- MVVM design pattern
- Material Design
- Kotlin
- Coroutines
- Room
- Navigation Components
- Lifecycle
- Glide
- Android Image Cropper
- Circle Image View
